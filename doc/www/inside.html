Komentarze wewnątrz kodu

Komentarz dla sekcji

  • Linia 1 - nazwa/tytuł sekcji
  • Linia 2 - 30 znaków =
  • Linia 3 - opcjonalne instrukcja, wskazówki jak używać

Przykład:

// FONTS
// ============================================================
// [i] --file should contain the names of the files in the specified --dir

Komentarz dla podsekcji (sekcja wewnątrz innej sekcji)

  • Linia 1 - nazwa/tytuł podsekcji
  • Linia 2 - 30 znaków -

Przykład:

// ROWS
// ------------------------------------------------------------
  • howto 1
  • howto 2
