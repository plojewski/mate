Mixins (from tollbox.scss)

see tollbox.scss on github

Media queries breakpoint

  • respond-to($point, $modifier: null)

Jeden z najważniejszych mixinów odpowiadający za media queries i breakpointy na potrzeby RWD.
Obsługowane breakpointy: mobile, phone, tablet, desktop

Mobile odpowiada za zakresy phone + tablet. Desktop może przyjąć dodatkowy argument xl,
przykład:

div[class*="col-"] { position: fixed; left: 0; top: 0; padding: $grid__row--gutter*2; @include respond-to(mobile) { position: static; padding: $grid__row--gutter; } } ul { text-align: right; @include respond-to(phone) { text-align: center; } } @include respond-to(desktop, xl) { width: 100%; }
