Mate to lekki, semantyczny framework na potrzeby front-endu.

Nazewnictwo klas CSS oparte jest na przestrzeni nazw bazującej na metodologii BEM (Block, Element, Modifier).

Bazowe funkcjonalności

  • 12 columns grid system
  • CSS class namespace based on BEM methodology
  • SASS
  • RWD
  • Web font icons
  • CSS & SCSS helpers library
