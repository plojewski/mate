Mate front-end framework

Mate to lekki, semantyczny framework (CSS i HTML5) na potrzeby front-endu. Podstawowym założeniem powstania Mate była prostota.
Zastosowaliśmy się do zasady "im prościej tym lepiej", dlatego

  • markup został maksymalnie uproszczony, nie stosujmey niepotrzbnych zagłębień kodu,
  • wszędzie gdzie to możliwe wykorzystujemy natywne możliwości HTML,
  • klasy CSS używane są tylko tam gdzie wymaga tego sytuacja, np. nie używamy klasy .form dla formualrzy, czy .table dla bazowych tabel.

Pozostałe założenia i funkcje:

  • Grid system
  • RWD
  • SASS
  • Namespace for CSS classes and SASS variables have been inpired by BEM
  • CSS & SCSS helpers + toolbox library
  • Web font icons from IcoMoon
