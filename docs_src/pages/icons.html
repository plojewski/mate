--- title: Icons ---

Icons come mainly from the library found at icomoon.io (most of them are Material Design Icons from Google).
The set of icons you can very easily expand their own positions. To make this possible we have included in the fonts/icomoon folder a configuration file mate.json exported from IconMoon, which you can use in IconMoon App.
